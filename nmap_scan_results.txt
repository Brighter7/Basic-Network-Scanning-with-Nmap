135/TCP (MSRPC): Windows Remote Procedure Call service.
MSRPC enables Windows applications and services to communicate with one another across a network. It is essential for system administration, Windows Management Instrumentation (WMI), and many Windows services. Frequently targeted by attackers for reconnaissance and remote exploitation. Should only be accessible to trusted hosts and protected by a firewall.


445/TCP (SMB): Windows file and printer sharing.
SMB (Server Message Block) is used for file sharing, printer sharing, and remote administrative tasks on Windows systems, and one of the most commonly attacked ports. Vulnerabilities such as EternalBlue (MS17-010) exploited SMB. Exposing this port unnecessarily can lead to ransomware or unauthorized access.
